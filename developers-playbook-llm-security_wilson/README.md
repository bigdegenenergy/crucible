# The Developer's Playbook for Large Language Model Security

**Author(s):** Steve Wilson

**Category:** AI Security

---

## Why This Book?

Written by the leader of the OWASP Top 10 for LLM Applications project. Covers prompt injection, data poisoning, supply chain attacks, and defensive strategies specific to LLMs.
