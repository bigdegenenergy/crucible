# Hands-On GPU Programming with Python and CUDA

**Author(s):** Dr. Brian Tuomanen

**Category:** GPU Computing

---

## Why This Book?

Covers PyCUDA, scikit-cuda, profiling with Nsight, CUDA libraries (cuBLAS, cuFFT), writing GPU kernels and device functions. Enables debugging and optimizing GPU code.
